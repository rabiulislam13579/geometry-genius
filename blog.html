Q 1: What is DOM and what is the purpose of the DOM?

Answer: DOM is document object model.At the time of loading a web page,the browser create a DOM.The html DOM is constructed as a tree model.
To add the whole html file add to javascript as object.Dom is used to add class, element and change the value of element.



Q 2: How will you select HTML elements using DOM? Name the DOM methods?

Answer:To select an element by using DOM selectors.
DOM methods are --
1.getElementsByTagName()
2.getElementsByClassName()
3.getElementById()
4.querySelector()
5.querySelectorAll()



Q 3: What is event bubble ?

Answer: When an event is directed to an element,and a event handler is set for element and the event handler is start then the event bubbles up to the elements parent.The event continues to bubble, until it reaches the top .



Q 4:What is a callback function and why will you use it?

Answer: A function is a block of code that perform a certain task when called. A call back function is define as the passed as argument to another function and call it later.
Call back function helps us to develop asynchronous JavaScript code and keeps us safe from problems and errors.
